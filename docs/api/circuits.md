# Circuits

The circuit pool and quantum-kernel computation. The seven bundled
encoding circuits are described in [Circuit Pool](../circuit_pool.md);
all are implemented in the bundled Qsun simulator.

Registering a custom circuit is a dict insertion - see
[Advanced Usage](../advanced_usage.md#2-custom-circuit-evaluator), and
mind the input-range warning on `UNIT_RANGE_CIRCUITS`.
